O que é HTML?

HTML significa "HyperText Markup Language", É a estrutura fundamental de uma página web, permitindo que organizemos texto, imagens e outros conteúdos.

O que é CSS?

CSS3 ou cascading style sheets level 3, é a terceira versão da linguagem de estilo CSS.

O CSS é usado para descrever a apresentação de um documento HTML e XML. Ele trás uma serie de recursos em relação ás versões anteriores, permitindo que os desenvolvedores criem interfaces mais ricas e interativas.

O que é JS?

JavaScript é uma linguagem de programação usada principalmente para deixar as páginas WEB interativas.

Com javaScript, você pode criar efeitos dinâmicos, responder a ação do usuário (como clicks em botões), validar formulários, carregar conte+udos sem precisar atualizar a página, enter muitas outras coisas.
